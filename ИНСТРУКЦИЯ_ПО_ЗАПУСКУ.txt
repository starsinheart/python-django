КАК ЗАПУСТИТЬ ПРОЕКТ:
====================

ВАРИАНТ 1 (Автоматический):
1. Запустите файл setup_project.bat
2. Следуйте инструкциям на экране
3. Создайте суперпользователя
4. Сервер запустится автоматически

ВАРИАНТ 2 (Ручной):
1. Откройте командную строку в папке проекта
2. Перейдите в папку STUD: cd STUD
3. Примените миграции: python manage.py migrate
4. Создайте суперпользователя: python manage.py createsuperuser
5. Запустите сервер: python manage.py runserver

ДОСТУП К САЙТУ:
===============
- Главная страница: http://127.0.0.1:8000/
- Админ-панель: http://127.0.0.1:8000/admin/
- Регистрация: http://127.0.0.1:8000/account/register/
- Вход: http://127.0.0.1:8000/account/login/

ПЕРВЫЕ ШАГИ:
============
1. Создайте суперпользователя (админа)
2. Войдите в админ-панель
3. Создайте группы студентов
4. Создайте дисциплины
5. Создайте оценки
6. Зарегистрируйте пользователей
7. Подтвердите их аккаунты в админ-панели
8. Свяжите студентов с пользователями
