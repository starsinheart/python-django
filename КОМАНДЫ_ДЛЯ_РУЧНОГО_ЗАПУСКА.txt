КОМАНДЫ ДЛЯ РУЧНОГО ЗАПУСКА ПРОЕКТА STUD
==========================================

Если автоматические скрипты не работают, выполните эти команды вручную:

1. Откройте командную строку (cmd)
2. Перейдите в папку проекта:
   cd "C:\Users\Ardor\Desktop\4 курс 7 семестр\программирование или DJANGO\PROG\STUD"

3. Удалите старую базу данных (если есть):
   del db.sqlite3

4. Примените миграции:
   python manage.py migrate

5. Создайте суперпользователя:
   python manage.py createsuperuser

6. Запустите сервер:
   python manage.py runserver

7. Откройте браузер и перейдите по адресу:
   http://127.0.0.1:8000/

ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ:
=============================

Проблема: "InconsistentMigrationHistory"
Решение: Удалите файл db.sqlite3 и выполните команды заново

Проблема: "No module named 'django'"
Решение: Активируйте виртуальное окружение:
         venv\Scripts\activate

Проблема: "Permission denied"
Решение: Запустите командную строку от имени администратора

Проблема: "Python not found"
Решение: Убедитесь, что Python установлен и добавлен в PATH

АЛЬТЕРНАТИВНЫЙ СПОСОБ:
=====================
1. Запустите файл ПРОСТОЙ_ЗАПУСК.bat
2. Следуйте инструкциям на экране
3. Введите данные для суперпользователя
4. Сервер запустится автоматически

ПОСЛЕ УСПЕШНОГО ЗАПУСКА:
========================
1. Создайте суперпользователя (админа)
2. Войдите в админ-панель /admin/
3. Создайте группы студентов
4. Создайте дисциплины и оценки
5. Зарегистрируйте пользователей
6. Подтвердите их аккаунты
7. Свяжите студентов с пользователями
