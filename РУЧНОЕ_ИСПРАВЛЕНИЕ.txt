РУЧНОЕ ИСПРАВЛЕНИЕ ПРОБЛЕМЫ С БАЗОЙ ДАННЫХ
==========================================

ПРОБЛЕМА: База данных заблокирована и содержит старые записи о миграциях

РЕШЕНИЕ 1 (Автоматическое):
1. Запустите файл ИСПРАВЛЕНИЕ_БАЗЫ_ДАННЫХ.bat
2. Следуйте инструкциям на экране

РЕШЕНИЕ 2 (Полное пересоздание):
1. Запустите файл ПОЛНОЕ_ПЕРЕСОЗДАНИЕ.bat
2. Следуйте инструкциям на экране

РЕШЕНИЕ 3 (Ручное):
1. Откройте командную строку от имени администратора
2. Перейдите в папку STUD
3. Выполните команды:

   taskkill /f /im python.exe
   taskkill /f /im pythonw.exe
   del db.sqlite3
   del account\migrations\*.py
   del my_site\migrations\*.py
   python manage.py makemigrations
   python manage.py migrate
   python manage.py createsuperuser
   python manage.py runserver

РЕШЕНИЕ 4 (Если ничего не помогает):
1. Закройте все программы (включая браузер)
2. Перезагрузите компьютер
3. Запустите ПОЛНОЕ_ПЕРЕСОЗДАНИЕ.bat

ПРИЧИНЫ ПРОБЛЕМЫ:
=================
- База данных заблокирована другим процессом
- Старые записи о миграциях в базе данных
- Конфликт между старой и новой системой пользователей

ЧТО ДЕЛАЮТ СКРИПТЫ:
===================
1. Останавливают все процессы Python
2. Удаляют базу данных
3. Удаляют все миграции
4. Создают новые миграции
5. Применяют миграции
6. Создают суперпользователя
7. Запускают сервер

ПОСЛЕ ИСПРАВЛЕНИЯ:
==================
1. Создайте суперпользователя
2. Войдите в админ-панель /admin/
3. Создайте группы студентов
4. Создайте дисциплины и оценки
5. Зарегистрируйте пользователей
6. Подтвердите их аккаунты
